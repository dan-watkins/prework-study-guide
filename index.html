HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The "UL" tag creates an Unordered List, or bullet points.
  • The "LI" tag creates line items, which is the text that appears in the bullet points from the previous step.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • External Style Sheets are an example of separation of concerns.
  • Class Selectors are a good way to add a single style to appropriately tagged elements at once.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch -d branch-name: deletes a branch you had previously created
  • git status shows the working tree (branch) status
  • git pull origin main pulls down the current version of the main repo into your current branch
  • it's important to remember to merge your PR's to main, otherwise your index.html file might disappear on you
