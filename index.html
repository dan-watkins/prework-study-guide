HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The "UL" tag creates an Unordered List, or bullet points.
  • The "LI" tag creates line items, which is the text that appears in the bullet points from the previous step.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch -d branch-name: deletes a branch you had previously created
  • git status shows the working tree (branch) status
  • git pull origin main pulls down the current version of the main repo into your current branch
  • it's important to remember to merge your PR's to main, otherwise your index.html file might disappear on you
